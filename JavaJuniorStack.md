# 2) Backend Java Júnior
#### Aqui você começa a fazer o que o mercado realmente espera de um júnior backend.

- Spring Boot
- Lombok
- Migrations
- Variáveis de ambiente
- Pagination and Sorting
- API REST:
  - controller
  - service
  - repository
  - DTO
  - request/response
- JPA / Hibernate
- Relacionamentos entre entidades:
  - 1:1
  - 1:N
  - N:N
- Validação de dados
- Tratamento centralizado de erros 
- HTTP:
  - verbos
  - status code
  - headers
- Autenticação com JWT
- Spring Security no básico 
- SQL intermediário
- Modelagem simples de banco
- Testes:
  - JUnit
  - Mockito básico
- Swagger / OpenAPI
- Clean Code no básico
- Estrutura de projeto organizada
- Deploy básico
- Docker básico
- Postman / Insomnia

O que precisa saber fazer nessa fase
- Criar API do zero
- Conectar API com banco
- Fazer CRUD com regra de negócio
- Fazer login com JWT
- Tratar erros de forma profissional
- Escrever testes básicos
- Subir projeto online
